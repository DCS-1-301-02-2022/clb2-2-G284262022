# 情報工学科
## コンピュータの基礎から、高度なプログラミングまで。
情報化社会をさらに進化させる、**IT業界の新たなリーダー**を育成。
## ３つのコース
1. システムエンジニアリングコース
1. インターネット＆セキュリティコース
1. 知能情報コース
## コラボレーションコース
- ビジュアルコンピューティングコース
- Webデザインコース
